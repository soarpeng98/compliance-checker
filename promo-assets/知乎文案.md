**一、绝对化用语（广告法第九条）**

以下词汇**绝对禁止**在广告中使用：

| 违禁词 | 建议替换 |
|--------|---------|
| 最好、最佳 | 优质、出色 |
| 第一、首个、首选 | 领先、推荐 |
| 唯一、独一无二 | 独特、独具特色 |
| 顶级、最高级 | 高端、优质 |
| 绝对、100%、百分百 | 非常、高比例 |
| 国家级、世界级 | 需有真实认证 |
| 全网第一、销量第一 | 销量领先、深受欢迎 |
| 王牌、冠军、之王 | 精品、优秀 |

**二、各行业专项违禁词**

**💄 美妆护肤**
- 禁止医疗宣称：消炎、抗敏、杀菌、排毒、治愈、再生
- 特殊用途需备案：祛斑、美白、防晒（需国妆特字）
- 平台禁用：刷酸、微针、水光、药妆、医美级

**🍎 食品保健品**
- 禁止医疗功效：治疗、抗癌、降血压、降血糖
- 禁止减肥宣称：减肥、燃脂、瘦身
- 严重违规：壮阳、排毒、延年益寿

**🏥 医疗健康**
- 绝对禁止：根治、包治、永不复发、无副作用
- 禁止神化：神医、祖传秘方、妙手回春
- 禁止数据：治愈率XX%、有效率XX%

**💰 金融理财**
- 禁止承诺：保本、保本保息、稳赚不赔
- 禁止诱导：躺着赚钱、一夜暴富、内幕消息
- 禁止绝对化：无风险、零风险、绝对安全

**📚 教育培训**
- 禁止承诺结果：保过、包过、包拿证、100%通过
- 禁止借名：命题组、出题人、教育部认证

**📱 直播带货**
- 违规词：全网最低价、史低价、破价、厂家直销（未经授权）
- 禁用营销话术：跳楼价、血亏、最后一天、再不买就没了

## 回答2：电商运营需要避开哪些坑？

**目标问题**："做电商/直播需要注意哪些违规词？""电商运营有哪些坑？"

**开篇**：
做了5年电商运营，踩过的坑比走过的路还多。分享几个最容易忽略的"文字雷区"：

**坑1：详情页违禁词**

很多卖家不知道，淘宝/拼多多/抖音都有**平台专属禁用词**，和广告法不完全一样：

- "原单""尾单""高仿""A货" → 直接违规
- "跳楼价""血亏""亏本清仓" → 平台禁用
- "专柜正品""假一赔十" → 需有授权证明，否则违规

**坑2：直播话术踩雷**

直播最容易出事的几个词：
- "全网最低" → 违反广告法
- "这个价格疯了" → 平台判定虚假营销
- "再不买就没了" → 威胁式营销，违规

**坑3：图片文字也要注意**

图片里的文字同样受监管！很多卖家只在详情页检查，忽略了主图、轮播图上的文字。
