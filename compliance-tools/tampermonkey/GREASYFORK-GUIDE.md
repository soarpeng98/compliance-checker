### 简短描述
```
一键检测网页文案中的广告违禁词。覆盖7大行业120+规则，Ctrl+Shift+C呼出面板，右下角🛡️浮动按钮随时检测。
```

### 详细描述（复制下面全部内容）
---

🛡️ **AI内容合规检测** 是一款全浏览器通用的违禁词扫描脚本。安装后，任意网页右下角会出现一个🛡️浮动按钮，点击即可呼出检测面板。

**🎯 核心功能**

- ⚡ **一键检测**：粘贴文案点击检测，秒级出结果
- 🎯 **7大行业规则库**：通用广告法、美妆护肤、食品保健、医疗健康、金融理财、教育培训、直播带货、电商通用
- 📊 **120+条违禁词**：每条规则标注违规原因和合规替换建议
- 🏷️ **分级提示**：🔴严重违规（绝对禁用）+ 🟡警告提示（建议修改）
- ⌨️ **快捷键**：`Ctrl+Shift+C` 随时呼出面板
- 🔒 **本地检测**：不会上传你的文案到任何服务器，绝对安全
- 💰 **完全免费**：无任何收费

**📊 覆盖行业**

| 行业 | 规则数 | 示例 |
|------|--------|------|
| ⚖️ 通用广告法 | 20条 | "最好""第一""国家级""包治百病" |
| 💄 美妆护肤 | 16条 | "消炎""再生""药妆""刷酸" |
| 🍎 食品保健 | 13条 | "抗癌""减肥""瘦身""壮阳" |
| 🏥 医疗健康 | 7条 | "根治""无副作用""祖传秘方" |
| 💰 金融理财 | 8条 | "保本""稳赚不赔""内幕消息" |
| 📚 教育培训 | 7条 | "保过""包拿证""命题组" |
| 🛒 电商通用 | 5条 | "高仿""A货""跳楼价" |
| 📱 直播带货 | 4条 | "全网最低""破价""史低价" |

**🖥️ 适用场景**

- 写公众号文章前，粘贴检测 → 避开广告法雷区
- 编辑淘宝详情页时，一键扫描 → 避开平台禁用词
- 准备直播脚本，复用文案检测 → 避开封号风险
- 审核外包文案，秒级扫描 → 节省人工时间

**📥 安装方法**

1. 确保已安装 [Tampermonkey](https://www.tampermonkey.net/) 扩展（Edge/Chrome/360/QQ浏览器均支持）
2. 点击本页面的「安装此脚本」按钮
3. 刷新任意网页，右下角出现 🛡️ 浮动按钮即可使用

**📢 使用技巧**

- 在任何网页选中文本 → 点击 🛡️ 按钮 → 文本自动填入检测面板
- 支持多个行业同时检测，点选行业标签即可
- 按 `Ctrl+Shift+C` 快速呼出/关闭面板
- 按 `Esc` 关闭面板
